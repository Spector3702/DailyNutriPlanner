# Backend

## Download Package (for running)
```
go mod download
```

## Download Pachage (for developing)
```
go mod download
go install github.com/swaggo/swag/cmd/swag
```

## Start Project
```
go run main.go
```

## Testing

### Install Ginkgo

```shell
go install github.com/onsi/ginkgo/v2/ginkgo
```

### Run Tests

```shell
ginkgo -r
```
